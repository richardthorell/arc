# ARC High-Level Tests

This folder is reserved for future high-level integration tests that exercise the engine, editor, and samples together.

Current low-level engine unit tests live under their engine modules, for example
`engine/simd/tests`, `engine/math/tests`, `engine/render/tests`, and
`engine/scene/tests`.

Future tests here should prefer full-engine/editor flows over testing individual
module internals.
